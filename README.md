## Getting Started

To get started follow the following steps

### Prerequisites
- npm
```npm install npm@latest -g```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/RamsesRomeroJr/HypeGeek.git
   ```
1. Install NPM packages
   ```sh
   npm install
   ```
